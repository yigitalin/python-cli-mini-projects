# Python CLI Mini Projects

A collection of beginner-friendly command-line projects built while learning Python.

---

## 📁 Projects

### 🎮 Rock Paper Scissors

* Player vs Computer gameplay
* Score tracking
* First to 3 or 5 wins
* Modular structure using functions

### 🏦 Banking Program

* Show balance
* Deposit money
* Withdraw money
* Menu-based navigation

### 🎰 Slot Machine

* 3x3 slot grid system
* Spin-based gameplay
* Betting system
* Balance tracking
* Multiple winning rows
* Payout system based on symbol matches

---

## 🧠 Purpose

This repository is focused on practicing core Python concepts:

* Functions
* Loops
* Conditionals
* User input handling
* Random module usage
* Basic project structure

---

## 🗂️ Project Structure

```text
python-cli-mini-projects/
├── rock-paper-scissors/
├── banking/
├── slot_machine/
```

---

## ▶️ How to Run

Navigate into any project folder and run:

```bash
python main.py
```
